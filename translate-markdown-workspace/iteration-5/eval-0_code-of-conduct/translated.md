# 贡献者公约 行为准则

## 我们的承诺

我们作为社区成员、贡献者和领导者承诺，让每个人都能以无骚扰的方式参与我们的社区，无论其年龄、体型、可见或不可见的残疾、种族、性征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外表、种族、宗教或性身份和取向如何。

我们承诺以有助于建立开放、友好、多元、包容和健康社区的方式行动和互动。

## 我们的标准

对我们的社区有所贡献的行为示例包括：

* 对他人表现出同情心和善意
* 尊重不同的意见、观点和经验
* 给予并优雅地接受建设性反馈
* 为自己的错误承担责任并向受影响的人道歉，以及从经验中学习
* 关注的不仅是对我们个人最有利的事项，也是对整个社区最有利的事项

不可接受的行为示例包括：

* 使用性化语言或意象，以及任何形式的性关注或示好
* 制造麻烦、侮辱性或贬低性评论，以及人身攻击或政治攻击
* 公开或私人骚扰
* 未经明确许可发布他人的私人信息，例如物理地址或电子邮件地址
* 其他在专业环境中合理被视为不当的行为

## 执法责任

社区领导者负责阐明和执行我们的行为标准，并将对他们认为不当、威胁性、冒犯性或有害的任何行为采取适当且公平的纠正措施。

社区领导者有权和责任删除、编辑或拒绝与本《行为准则》不符的评论、提交、代码、wiki 编辑、问题和其他贡献，并在适当时沟通审核决定的原因。

## 范围

本《行为准则》适用于所有社区空间，也适用于个人以正式代表身份代表社区出现在公共空间时。代表我们社区的示例包括使用官方电子邮件地址、通过官方社交媒体账户发布或担任在线或线下活动的指定代表。

## 执法

滥用、骚扰或其他不可接受行为的情况可向负责执法的社区领导者报告，电子邮件地址为 jesse@primeradiant.com。所有投诉将被及时公正地审查和调查。

所有社区领导者都有义务尊重任何事件报告者的隐私和安全。

## 执法指南

社区领导者在确定任何他们认为违反本《行为准则》的行为的后果时，将遵循以下社区影响指南：

### 1. 纠正

**社区影响**：使用不当语言或其他被认为在社区中不专业或不受欢迎的行为。

**后果**：社区领导者发出的私人书面警告，阐明违反性质并解释该行为为何不当。可能会要求进行公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列行为造成的违反。

**后果**：警告及因持续行为而产生的后果。在规定时间内不与相关人员互动，包括与执行本《行为准则》的人不请自来的互动。这包括避免在社区空间以及社交媒体等外部渠道中的互动。违反这些条款可能导致临时或永久禁令。

### 3. 临时禁令

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在规定时间内临时禁止与社区进行任何形式的互动或公开交流。在此期间，禁止与相关人员进行任何公开或私人互动，包括与执行本《行为准则》的人不请自来的互动。违反这些条款可能导致永久禁令。

### 4. 永久禁令

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、对个人的骚扰或对个人群体的侵犯或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。
