# 贡献者契约行为守则

## 我们的承诺

我们作为成员、贡献者和领导者承诺，为我们社区的参与创造一个
无骚扰的体验，无论年龄、身体大小、可见或不可见的残疾、种族、性征特征、性别
认同和表达、经验水平、教育、社会经济地位、
国籍、个人外表、种族、宗教或性身份
和性取向。

我们承诺以有助于营造开放、热烈、
多样、包容和健康社区的方式行动和互动。

## 我们的标准

有助于为我们社区营造积极环境的行为示例包括：

* 对他人表现出同情和友善
* 尊重不同的观点、观念和经验
* 接受和优雅地接纳建设性反馈
* 接受责任并向受我们错误影响的人道歉，
  并从体验中学习
* 关注对所有人以及整个社区而不仅仅是对个人最有利的事项

不可接受行为的示例包括：

* 使用性化语言或意象，以及任何形式的性关注或
  性骚扰
* 钓鱼、侮辱或贬低性评论，以及个人或政治攻击
* 公开或私下骚扰
* 在未获得明确许可的情况下发布他人的私人信息，例如物理
  地址或电子邮件地址
* 在专业环境中可能合理地被认为不恰当的其他行为

## 执行责任

社区领导者负责澄清和执行我们的可接受行为标准，
并将对任何他们认为不恰当、威胁、冒犯的行为采取适当和公平的纠正措施
或有害。

社区领导者有权和责任删除、编辑或拒绝
不符合本行为守则的评论、提交、代码、wiki编辑、议题和其他贡献，
并在适当时说明审核决定的原因。

## 适用范围

本行为守则适用于所有社区空间，也适用于
个人在公共场所正式代表社区时。
代表我们社区的示例包括使用官方电子邮件地址、
通过官方社交媒体账户发布，或在在线或离线活动中担任指定
代表。

## 执行

滥用、骚扰或其他不可接受行为的事例可能会
报告给负责执行的社区领导者，地址为
jesse@primeradiant.com。
所有投诉将被迅速和公平地审查和调查。

所有社区领导者都有义务尊重
任何事件报告者的隐私和安全。

## 执行指南

社区领导者将遵循这些社区影响指南来确定
对他们认为违反本行为守则的任何行为的后果：

### 1. 纠正

**社区影响**：在社区中使用不恰当的语言或其他被认为
不专业或不受欢迎的行为。

**后果**：来自社区领导者的私下书面警告，
提供有关违反性质的清晰说明和
关于为什么该行为不恰当的解释。可能会要求做出公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列
行为进行的违反。

**后果**：警告伴随持续行为的后果。不
与涉及的人进行互动，包括与强制执行本行为守则的人员的非主动互动，
在指定时间内。这包括避免在社区空间以及社交媒体等外部渠道中的互动。
违反这些条款可能导致临时或
永久禁止。

### 3. 暂时禁止

**社区影响**：严重违反社区标准，包括
持续的不恰当行为。

**后果**：在指定时间内暂时禁止与社区进行任何形式的互动或公开
沟通。在此期间不允许公开或
与涉及的人员进行私下互动，包括与强制执行本行为守则的人员的非主动互动。
违反这些条款可能导致永久禁止。

### 4. 永久禁止

**社区影响**：表明违反社区
标准的模式，包括持续的不恰当行为，对个人的骚扰，
或对个人阶层的侵犯或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。
