# 前端设计插件

生成独特、生产级别的前端界面，避免通用人工智能审美。

## 功能介绍

Claude 会自动在前端工作中使用此技能(skill)。创建生产就绪的代码，具有：

- 大胆的美学选择
- 独特的排版和配色方案
- 高影响力的动画和视觉细节
- 上下文感知的实现

## 使用示例

```
"Create a dashboard for a music streaming app"
"Build a landing page for an AI security startup"
"Design a settings panel with dark mode"
```

Claude 将选择清晰的美学方向，并实现生产代码，注重细节。
